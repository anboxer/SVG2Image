SVG2Image
=========

export svg to image file.

Usage
=========
```
svg2image({
	src: svg,
	dest: '',
	width: 100,
	height: 400
});
```
the sample is [here](src/index.html)

### option.src
svg DOM element

### option.dest
name of the image, default is ISO Date

### option.width
image's width

### option.height
image's height
