SVG2Image
=========

export svg to image file. <br/>
NOTE: This project hasn't be tested

Usage
=========
svg2image({<br/>
	src: svg,<br/>
	dest: '',<br/>
	width: 100,<br/>
	height: 400<br/>
});<br/>
please see src/index.html
###option.src
svg DOM element

###option.dest
name of image, default is ISO Date formater

###option.width
image's width

###option.height
image's height
